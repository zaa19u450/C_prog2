#Тесты для лабораторной работы №8

## Негативные тесты:

### Ошибки командной строки

(для этих тестов неважны входные данные)

- 01 - недостаточно аргументов (<4);
- 02 - слишком много аргументов (>5);
- 03 - неверное значение `action`;
- 04 - для команды `a` недостаточно аргументов (4);
- 05 - для команды `m` недостаточно аргументов (4);
- 06 - для команды `o` слишком много аргументов (5);


### Ошибки в файле 

(для этих тестов даны только первые входные файлы,
второй в случае необходимости обрабатывается аналогично>
алгоритм считывания данных из файла для всех команд одинаковый)

- 07 - файл не существует (для этого теста входного файла нет);
- 08 - файл пустой;
- 09 - неверное количество строк n (<=0);
- 10 - нецелое (аналогично обрабатывается нечисловое) количество строк n;
- 11 - неверное количество столбцов m (<=0);
- 12 - нецелое (аналогично обрабатывается нечисловое) количество столбцов m;
- 13 - отрицательное количество ненулевых элементов;
- 14 - слишком большое количество ненулевых элементов (> n*m);
- 15 - нецелое (аналогично обрабатывается нечисловое) количество ненулевых элементов (> n*m);
- 16 - в записи параметров и значения ненулевого элемента матрицы недостаточно записей (< 3);
- 17 - в записи параметров и значения ненулевого элемента матрицы введено нечисловое значение;
- 18 - слишком маленький номер строки (<1)
- 19 - слишком большой номер строки (>n)
- 20 - слишком маленький номер столбца (<1)
- 21 - слишком большой номер столбца (>m)
- 22 - недостаточно записей (<числа ненулевых элементов)
- 23 - слишком много записей (>числа ненулевых элементов)
- 24 - в записи ненулевого элемента введен нулевой
- 25 - в записях ненулевых элементов на одно и то же место пытаются записать значение больше одного раза

### Ошибки в размерах обрабатываемых матриц
- 26 - для команды сложения размеры матриц не совпадают
- 27 - для команды умножения матриц не совпадают количество столбцов в первой матрице и количество строк во второй
- 28 - для команды вычисления определителя методом Гаусса дана не квадратная матрица


## Позитивные тесты:

### Команда сложения

- 01 - сложение с нулевой матрицей;
- 02 - все элементы новой матрицы получаются нулевыми (или меньше эпсилон);
- 03 - обычный тест (получаются нулевые, положительные и отрицательные элементы);

### Команда умножения 
- 04 - умножение на единичную матрицу (первая и вторая матрицы - квадратрные, равных размеров);
- 05 - умножение на нулевую матрицу (матрицы согласованы, но не одинакового размера);
- 06 - обычный тест (получаются нулевые, положительные и отрицательные элементы);

### Команда вычисления определителя методом Гаусса
- 07 - матрица вырожденная: все элементы одного ряда нулевые;
- 08 - матрица вырожденная: 2 ряда пропорциональны друг другу;
- 09 - обычный тест (для вычисления определителя методом Гаусса требуются перестановки, включая перестановку первой строки);
